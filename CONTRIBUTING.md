# Contributing

Thank you for your interest in Azure Functions Python DX Toolkit.

## How to contribute

This is a hub repository that connects multiple focused packages. Contributions are welcome in several forms:

### Feedback

The most valuable contribution right now is **feedback**.

- What Azure Functions Python pain points do you face?
- Which tools are most useful to you?
- What's missing from the toolkit?

Please [open a feedback issue](https://github.com/yeongseon/azure-functions-python-dx/issues/new?template=feedback.yml).

### Bug reports

If you find a bug in a specific package, please open an issue in that package's repository:

### Documentation improvements

- Typo fixes, clarifications, and better examples are always welcome.
- Open a pull request against the relevant repository.

### Other contributions

- Real-world use cases and integration ideas
- Suggestions for missing DX features
- Comparison notes with other tools or frameworks
